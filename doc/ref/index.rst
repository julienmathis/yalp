Plugin Reference
================

YALP uses plugins for the three components.

:doc:`Inputers </ref/inputers/all/index>`
    Collect events from input sources, such as log files.

:doc:`Parsers </ref/parsers/all/index>`
    Process a raw input event and transform, extract, or modify the input into
    a more organized output.

:doc:`Outputers </ref/outputers/all/index>`
    Record the event into a persistant storage, such as a file or mongo
    database.
